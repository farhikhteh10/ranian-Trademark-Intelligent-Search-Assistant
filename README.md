# ranian-Trademark-Intelligent-Search-Assistant
این نرم‌افزار یک ابزار پیشرفته **هوش تجاری (BI)** جهت اتوماسیون، آنالیز و استعلام وضعیت ثبت برند در سامانه مرکز مالکیت معنوی ایران است. برخلاف ربات‌های ساده، این سیستم مجهز به یک هسته پردازشی ۴ لایه است که رفتار کارشناسان اداره ثبت را شبیه‌سازی کرده و تعارضات پنهان برند را کشف می‌کند.
